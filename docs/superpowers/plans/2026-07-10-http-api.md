# HTTP API 自动化交互系统 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 LiteTerm 添加本地 HTTP API（axum），使 Claude Code 等外部工具能通过 `curl` 程序化操作终端标签页（打开/关闭/写入/读取输出）。

**Architecture:** Tauri 进程内嵌 axum HTTP 服务器（127.0.0.1:19526），与 Tauri 后端共享 `Arc<AppState>`。每个终端的 PTY reader 线程在 emit 给前端的同时，额外写入一个环形缓冲区（1MB）。HTTP API 通过游标提供增量读取。前端通过 Tauri event 监听 API 触发的标签页变更。

**Tech Stack:** Rust (axum, tokio, strip-ansi-escapes, rand), TypeScript/React (Tauri event listener)

## Global Constraints

```rust
pub struct TerminalOutputBuffer {
    buf: Vec<u8>,
    capacity: usize,
    head: usize,
    write_pos: u64,
}

impl TerminalOutputBuffer {
    pub fn new(capacity: usize) -> Self {
        Self {
            buf: vec![0u8; capacity],
            capacity,
            head: 0,
            write_pos: 0,
        }
    }

    pub fn write(&mut self, data: &[u8]) {
        for &byte in data {
            self.buf[self.head] = byte;
            self.head = (self.head + 1) % self.capacity;
        }
        self.write_pos += data.len() as u64;
    }

    /// 从 cursor 位置读取增量数据。返回 (data, new_cursor, truncated)。
    pub fn read_from(&self, cursor: u64) -> (Vec<u8>, u64, bool) {
        if cursor >= self.write_pos {
            return (Vec::new(), self.write_pos, false);
        }

        let available = self.write_pos - cursor;
        let truncated = available > self.capacity as u64;

        let read_start = if truncated {
            self.write_pos - self.capacity as u64
        } else {
            cursor
        };

        let len = (self.write_pos - read_start) as usize;
        let mut data = Vec::with_capacity(len);

        let start_offset = self.capacity - (self.write_pos - read_start) as usize % self.capacity;
        let start_idx = (self.head + start_offset) % self.capacity;

        for i in 0..len {
            data.push(self.buf[(start_idx + i) % self.capacity]);
        }

        (data, self.write_pos, truncated)
    }
}

#[derive(Clone, serde::Serialize)]
pub struct TabInfo {
    pub id: String,
    pub label: String,
    pub tab_type: String,
}
```

```rust
use std::sync::Arc;

pub struct ApiState {

    pub app_handle: AppHandle,
    pub token: String,

fn check_auth(headers: &HeaderMap, token: &str) -> Result<(), (StatusCode, Json<serde_json::Value>)> {
    let auth = headers
        .get("authorization")
        .and_then(|v| v.to_str().ok())
        .unwrap_or("");
    let expected = format!("Bearer {}", token);
    if auth != expected {
        return Err((
            StatusCode::UNAUTHORIZED,
            Json(serde_json::json!({"error": "未授权"})),
        ));
    }
    Ok(())
}

async fn list_tabs(
    AxumState(api): AxumState<ApiState>,
    headers: HeaderMap,
) -> Result<Json<serde_json::Value>, (StatusCode, Json<serde_json::Value>)> {
    check_auth(&headers, &api.token)?;
    let registry = api.app_state.tab_registry.lock().unwrap();
    let tabs: Vec<&TabInfo> = registry.values().collect();
    Ok(Json(serde_json::to_value(tabs).unwrap()))
}

pub async fn start_api_server(app_state: Arc<AppState>, app_handle: AppHandle) {
    // 生成 token
    use rand::Rng;
    let token_bytes: [u8; 32] = rand::thread_rng().gen();
    let token = hex::encode(token_bytes);

    // 写入 token 文件
    let config_dir = dirs::config_dir()
        .unwrap_or_else(|| dirs::home_dir().unwrap_or_default().join(".config"))
        .join("guishell");
    let _ = std::fs::create_dir_all(&config_dir);
    let token_path = config_dir.join("api_token");
    if let Err(e) = std::fs::write(&token_path, &token) {
        app_log!("API", "写入 token 文件失败: {}", e);
        return;
    }
    // 设置文件权限 0600
    #[cfg(unix)]
    {
        use std::os::unix::fs::PermissionsExt;
        let _ = std::fs::set_permissions(&token_path, std::fs::Permissions::from_mode(0o600));
    }
    app_log!("API", "token 已写入 {}", token_path.display());

    let api_state = ApiState {
        app_state,
        app_handle,
        token,
    };

    let app = Router::new()
        .route("/api/v1/tabs", get(list_tabs))
        .route("/api/v1/tabs/local", post(open_local_tab))
        .route("/api/v1/tabs/ssh", post(open_ssh_tab))
        .route("/api/v1/tabs/{id}/focus", put(focus_tab))
        .route("/api/v1/tabs/{id}/write", post(write_tab))
        .route("/api/v1/tabs/{id}/read", get(read_tab))
        .route("/api/v1/tabs/{id}", delete(close_tab))
        .with_state(api_state);

    let addr = "127.0.0.1:19526";
    let listener = match tokio::net::TcpListener::bind(addr).await {
        Ok(l) => l,
        Err(e) => {
            app_log!("API", "HTTP 服务器绑定 {} 失败: {}（API 不可用，不影响正常使用）", addr, e);
            return;
        }
    };
    app_log!("API", "HTTP 服务器已启动: {}", addr);

    if let Err(e) = axum::serve(listener, app).await {
        app_log!("API", "HTTP 服务器异常退出: {}", e);
    }
}
```

```rust
#[derive(Clone)]
pub struct ApiState {
    pub app_handle: AppHandle,
    pub token: String,
}
```

handler 中获取 state：

```rust
let state = api.app_handle.state::<AppState>();
```

这样完全不需要改现有 AppState 的字段类型，只需要加两个新字段。

**但是** `output_buffers` 需要在 reader 线程中写入，reader 线程拿不到 AppHandle... 

reader 线程在 terminal.rs / ssh.rs 中，它们通过 `State<'_, AppState>` 获取 state。`State::inner()` 返回 `&AppState`。如果 `output_buffers` 是 `Arc<Mutex<HashMap<...>>>`，可以 `.clone()` Arc 传进线程。如果是普通 `Mutex<HashMap<...>>`，不行。

所以 **只需要把 `output_buffers` 改成 `Arc<Mutex<...>>`**。`tab_registry` 不需要传进线程，可以保持普通 `Mutex`。

**最终决定：**
- `output_buffers`: `Arc<Mutex<HashMap<String, TerminalOutputBuffer>>>` — reader 线程要 clone Arc
- `tab_registry`: `Mutex<HashMap<String, TabInfo>>` — 只在 Tauri command 和 HTTP handler 中用，不需要传进线程
- `start_api_server` 签名: `pub async fn start_api_server(app_handle: AppHandle)`
- `ApiState` 持有 `AppHandle` + `token`
- handler 通过 `app_handle.state::<AppState>()` 获取 state

```rust
    crate::commands::api_server::start_api_server(app_handle).await;
```

```rust
pub fn do_open_terminal(
    state: &AppState,
    app: &AppHandle,
    shell_path: Option<String>,
) -> Result<String, String> {
    let id = uuid::Uuid::new_v4().to_string();
    let pty_system = portable_pty::native_pty_system();
    let pair = pty_system
        .openpty(PtySize { rows: 24, cols: 80, pixel_width: 0, pixel_height: 0 })
        .map_err(|e| { app_log!("TERM", "ERROR: PTY open failed: {}", e); e.to_string() })?;

    let shell = shell_path.unwrap_or_else(default_shell);
    let cmd = CommandBuilder::new(shell.clone());
    let _child = pair.slave.spawn_command(cmd).map_err(|e| {
        app_log!("TERM", "ERROR: spawn shell failed: {} (shell={})", e, shell);
        e.to_string()
    })?;
    drop(pair.slave);

    let reader = pair.master.try_clone_reader().map_err(|e| { app_log!("TERM", "ERROR: clone reader failed: {}", e); e.to_string() })?;
    let writer = pair.master.take_writer().map_err(|e| { app_log!("TERM", "ERROR: take writer failed: {}", e); e.to_string() })?;

    let (input_tx, input_rx) = std::sync::mpsc::channel::<Vec<u8>>();

    // 初始化输出缓冲区
    state.output_buffers.lock().unwrap().insert(id.clone(), crate::state::TerminalOutputBuffer::new(1_048_576));
    let output_bufs = state.output_buffers.clone();

    let id_clone = id.clone();
    let app_clone = app.clone();
    std::thread::spawn(move || {
        std::thread::sleep(std::time::Duration::from_millis(500));
        let mut reader = reader;
        let mut read_buf = [0u8; 4096];
        loop {
            match reader.read(&mut read_buf) {
                Ok(0) => break,
                Ok(n) => {
                    let _ = app_clone.emit(
                        "terminal-output",
                        serde_json::json!({"id": id_clone, "data": &read_buf[..n]}),
                    );
                    if let Ok(mut bufs) = output_bufs.lock() {
                        if let Some(ob) = bufs.get_mut(&id_clone) {
                            ob.write(&read_buf[..n]);
                        }
                    }
                }
                Err(_) => break,
            }
        }
    });

    std::thread::spawn(move || {
        let mut writer = writer;
        while let Ok(data) = input_rx.recv() {
            let _ = writer.write_all(&data);
            let _ = writer.flush();
        }
    });

    let (resize_tx, resize_rx) = std::sync::mpsc::channel::<(u32, u32)>();
    let master = pair.master;
    std::thread::spawn(move || {
        while let Ok((cols, rows)) = resize_rx.recv() {
            let _ = master.resize(PtySize { rows: rows as u16, cols: cols as u16, pixel_width: 0, pixel_height: 0 });
        }
    });

    state.local_terminals.lock().unwrap().insert(id.clone(), LocalTerminal {
        id: id.clone(),
        input_tx,
        resize_tx,
        stop: std::sync::Arc::new(std::sync::atomic::AtomicBool::new(false)),
    });

    Ok(id)
}
```

```rust
async fn open_ssh_tab(
    AxumState(api): AxumState<ApiState>,
    headers: HeaderMap,
    Json(body): Json<serde_json::Value>,
) -> Result<Json<serde_json::Value>, (StatusCode, Json<serde_json::Value>)> {
    check_auth(&headers, &api.token)?;

    let host = body.get("host").and_then(|v| v.as_str())
        .ok_or_else(|| (StatusCode::BAD_REQUEST, Json(serde_json::json!({"error": "缺少必填字段: host"}))))?
        .to_string();
    let user = body.get("user").and_then(|v| v.as_str())
        .ok_or_else(|| (StatusCode::BAD_REQUEST, Json(serde_json::json!({"error": "缺少必填字段: user"}))))?
        .to_string();
    let port = body.get("port").and_then(|v| v.as_u64()).unwrap_or(22) as u16;
    let password = body.get("password").and_then(|v| v.as_str()).map(|s| s.to_string());
    let auth_method = body.get("auth_method").and_then(|v| v.as_str()).unwrap_or("password").to_string();
    let key_path = body.get("key_path").and_then(|v| v.as_str()).map(|s| s.to_string());
    let proxy_jump = body.get("proxy_jump").and_then(|v| v.as_str()).map(|s| s.to_string());

    let label = format!("{}@{}", user, host);
    let state = api.app_handle.state::<AppState>();

    // 调用现有 ssh_connect 逻辑 — 但 ssh_connect 是 Tauri command，签名带 State<'_>
    // 直接通过 invoke 会很复杂，更好的做法是提取核心函数
    // 但 ssh_connect 非常复杂（500 行），不值得重构
    // 改用 tauri::command::CommandArg 手动调用 — 不可行
    // 最务实的做法：在 HTTP handler 中直接复用 AppHandle.state() + 调核心逻辑
    
    // 由于 ssh_connect 的逻辑非常复杂（ProxyJump/直连两条路径、ZMODEM、OSC7 钩子等），
    // 不提取核心函数。改用 AppHandle 上的 invoke 风格调用：
    // Tauri 2 不支持从 Rust 侧直接 invoke command，所以用前端事件 + emit 模式。
    
    // 实际可行方案：HTTP handler emit 一个事件给前端，前端收到后调用现有的 handleConnect
    // 但这样 HTTP 调用变成异步的了...
    
    // 最终方案：从 ssh.rs 提取 do_ssh_connect 内部函数（不带 #[tauri::command]），
    // 和 terminal.rs 的 do_open_terminal 一样的模式。
    // ssh_connect 调 do_ssh_connect，HTTP handler 也调 do_ssh_connect。
    
    let id = crate::commands::ssh::do_ssh_connect(
        &state, &api.app_handle,
        host.clone(), port, user.clone(), password, auth_method, key_path, label.clone(), proxy_jump,
        None, None, // cols, rows — HTTP 调用不知道前端尺寸，用默认值
    ).await
    .map_err(|e| (StatusCode::INTERNAL_SERVER_ERROR, Json(serde_json::json!({"error": e}))))?;

    // 注册
    state.tab_registry.lock().unwrap().insert(
        id.clone(),
        TabInfo { id: id.clone(), label: label.clone(), tab_type: "ssh".to_string() },
    );

    // 通知前端
    let _ = api.app_handle.emit("api-tab-opened", serde_json::json!({
        "id": id, "label": label, "type": "ssh",
        "sshParams": {
            "host": host, "port": port, "user": user
        }
    }));

    Ok(Json(serde_json::json!({"id": id, "label": label})))
}
```

```rust
async fn write_tab(
    AxumState(api): AxumState<ApiState>,
    headers: HeaderMap,
    Path(id): Path<String>,
    Json(body): Json<serde_json::Value>,
) -> Result<Json<serde_json::Value>, (StatusCode, Json<serde_json::Value>)> {

            (StatusCode::INTERNAL_SERVER_ERROR, Json(serde_json::json!({"error": e.to_string()})))
        })?;
        return Ok(Json(serde_json::json!({"ok": true})));
    }
    Err((StatusCode::NOT_FOUND, Json(serde_json::json!({"error": "terminal not found"}))))
```

```rust
async fn read_tab(
    AxumState(api): AxumState<ApiState>,
    headers: HeaderMap,
    Path(id): Path<String>,
    Query(params): Query<std::collections::HashMap<String, String>>,
) -> Result<Json<serde_json::Value>, (StatusCode, Json<serde_json::Value>)> {
    check_auth(&headers, &api.token)?;

    let cursor: u64 = params.get("cursor")
        .and_then(|c| c.parse().ok())
        .unwrap_or(0);
    let raw = params.get("raw")
        .map(|r| r == "true")
        .unwrap_or(false);

    let state = api.app_handle.state::<AppState>();
    let bufs = state.output_buffers.lock().unwrap();
    let buf = bufs.get(&id).ok_or_else(|| {
        (StatusCode::NOT_FOUND, Json(serde_json::json!({"error": "terminal not found"})))
    })?;

    let (data_bytes, new_cursor, truncated) = buf.read_from(cursor);

    let data_str = if raw {
        String::from_utf8_lossy(&data_bytes).to_string()
    } else {
        let stripped = strip_ansi_escapes::strip(&data_bytes);
        String::from_utf8_lossy(&stripped).to_string()
    };

    Ok(Json(serde_json::json!({
        "data": data_str,
        "cursor": new_cursor,
        "truncated": truncated,
    })))
}
```

```rust
    // 通知前端
    let _ = api.app_handle.emit("api-tab-closed", serde_json::json!({"id": id}));

    Ok(Json(serde_json::json!({"ok": true})))
```
